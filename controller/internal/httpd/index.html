Restore

Load a backup JSON: adopts its controller key (replay counter re-seeds so it clears each node's floor) and restores node aliases/positions. Applies live to this running controller. On a brand-new install with no key, use the CLI instead: agnctl -restore backup.json.

Membership

Each node mints its own Ed25519 key and derives its id from it (self-certifying). The gateway verifies every announce signature; the controller only manages (tunes/commands) nodes whose verified pubkey you've approved here. Approval is keyed on the pubkey, so spoofing an id can't grant membership.
