agnostic-LoRa-Net — web tools

Zero-install, in-browser tools for an app-agnostic LoRa mesh. Pick one:

⚡

Flash & provision a node →

Install the firmware on a board over USB (in-browser DFU), set the BLE PIN, and bind a controller key. Start here with a new board.
💬

Chat demo →

Flashed two or more nodes? Message across the mesh in clear text from your browser to confirm it works. The quickest end-to-end test.
🗺️

Live map →

See nodes on real geography with per-direction link quality, asymmetry, SNR margin and battery — plus a gateway console.
🛠️

Node manager →

Configure a connected node: radio PHY, BLE pairing/PIN, battery calibration, mobile flag, raw console — over USB or BLE.
Needs Chrome or Edge. These tools use Web Serial / Web Bluetooth, which aren't available in Firefox or Safari. The page must be served over HTTPS (this one is) or from localhost — a browser requirement, not ours.

Running a managed deployment? The agnctl controller serves a consolidated dashboard (map · decision feed · configure · flash) that supersedes these standalone apps. See the project README.
