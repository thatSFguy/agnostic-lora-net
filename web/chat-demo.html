LoRa mesh — clear-text chat demo

Send a message from your browser, across the LoRa radio mesh, to another browser. No internet, no cell network — just the radios.

💻 your browser→ BLE → 📟 node→ 📡 LoRa → 📟 node→ BLE →💻 their browser
First time? Setup in 3 steps
  1. Flash two (or more) nodes with the commissioning hub. On each, enable Bluetooth: connect over USB and run ble on, then note its 6-digit PIN (blepin).
  2. Open this page in two browser tabs (Chrome or Edge). In each tab press Connect, pick a different node (they show up as ALN-…), and enter the PIN.
  3. Each tab shows its own node id once connected. Copy one node's id into the other tab's “To” box, type a message, and hit Send. It should pop up in the other tab.

⚠️ Clear text — think telnet, not SSH. Messages go out unencrypted over the air, so anyone in radio range can sniff them, just like telnet on a shared wire. That's fine here: this demo exists to exercise the mesh, not to carry private traffic. When you need secured comms, run a Reticulum app over the node — it gives you the end-to-end encryption layer this demo deliberately leaves off (see docs/tcp-bridge.md).
