TX (teal) = how well a node is heard. RX (amber) = how well it hears back. Where they diverge, the link is asymmetric.

Announce model follows Mark Qvist's description of Reticulum path selection: the destination's announce propagates outward and each node keeps the path the most-recent announce arrived on fastest (no SNR / bandwidth metric). A reply rides the exact reverse of that path — so a one-way hop kills the two-way Link and forces rediscovery of a fully symmetric path, rather than borrowing a different return route.
